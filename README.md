# Incubyte TDD Assessment

Task Details: https://blog.incubyte.co/blog/tdd-assessment/

#### System Dependencies
* Ruby 3.2.6

#### Instructions
* Clone the git repository
* Run the command `gem install rspec`
* Run the command `rspec` to run all of the test cases
